<ToolDescription>

<ToolName>
Calculator
</ToolName>

<ToolUse>
The Calculator tool lets you compute literal mathematical expressions and get an exact answer to improve your accuracy on computation tasks
To use this tool, you have 2 modes : 
- compute : simply put any mathematical expression on each line of the <in_data></in_data> section, like so and they will be computed and their result returned to you :

<call>
<tool>
Calculator
</tool>
<action>
compute
</action>
<in_data>
9 + 10
4*6 + (45/7)
</in_data>
</call>


- check : check the validity of a comparison between 2 mathematical expressions on each line (available comparators : [>, <, =]), like so : 

<call>
<tool>
Calculator
</tool>
<action>
check
</action>
<in_data>
15 - 6 < 11 + 2
50 = 30
</in_data>
</call>

use the Calculator tool whenever you need to perform any kind of non-trivial literal math. Pay attention to the order of operations to be sure to compute what you intend to.
</ToolUse>

<ToolOutput>
This tool outputs the results of each computation passed to it, or errors if any can't be computed, a normal <output_data></output_data> section of the output for the first example input looks like this :

<output_data>
9 + 10 = 19
4*6 + (45/7) = 30.42857
</output_data>

for "check" actions, the output will look like this for the second example input : 

<output_data>
15 - 6 < 11 + 2 -> TRUE
50 = 30 -> FALSE
</output_data>

</ToolOutput>

<ToolErrors>
The following errors will appear when applicable :
- INCORRECT : an expression was formatted incorrectly
</ToolErrors>

</ToolDescription>
