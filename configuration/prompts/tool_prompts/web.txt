<ToolUse>
The Web tool lets you 
There are 2 different actions you can perform with this tool : 
- search : put the number of results you want and a query within quotes, one at a time on each line, like so : 

<action>
search
</action>
<in_data>
2 "Who is the current prime minister of Canada ?"
10 "What's the most common bird species in Turkey ?"
</in_data>

You can use the number of results parameter to manage the amount of information you are fetching. facts about very popular subjects should need few results to get a conclusive, trustable answer, whereas niche questions may need upwards of 5 to 20 results to be able to start to answer them.


- open : get the HTML code of one or more web pages as URLs, like so : 

<action>
open
</action>
<in_data>
https://example.com
https://en.wikipedia.org/wiki/Main_Page
</in_data>

use the Web tool whenever you need to get precise information on a recent subject 
</ToolUse>
